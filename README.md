
# ERP Gráfica

Sistema simples de gestão para gráficas.

Funcionalidades:
- Orçamentos
- Simulador de lona
- Cálculo de chapa MDF/Acrílico
- Kanban de produção

## Como publicar

1. Crie um repositório no GitHub
2. Envie todos os arquivos deste projeto
3. Vá em Settings → Pages
4. Ative: Deploy from branch → main

O sistema funcionará direto no navegador.
